struct tda10071_platform_data
=============================

.. c:type:: struct tda10071_platform_data

    Platform data for the tda10071 driver



Definition
----------

.. code-block:: c

  struct tda10071_platform_data {
    u32 clk;
    u16 i2c_wr_max;
    #define TDA10071_TS_SERIAL        0
    #define TDA10071_TS_PARALLEL      1
    u8 ts_mode;
    bool spec_inv;
    u8 pll_multiplier;
    u8 tuner_i2c_addr;
    struct dvb_frontend* (* get_dvb_frontend) (struct i2c_client *);
  };



Members
-------

:``u32 clk``:
    Clock frequency.

:``u16 i2c_wr_max``:
    Max bytes I2C adapter can write at once.

:``u8 ts_mode``:
    TS mode.

:``bool spec_inv``:
    Input spectrum inversion.

:``u8 pll_multiplier``:
    PLL multiplier.

:``u8 tuner_i2c_addr``:
    CX24118A tuner I2C address (0x14, 0x54, ...).

:``struct dvb_frontend* (*)(struct i2c_client *) get_dvb_frontend``:
    Get DVB frontend.
