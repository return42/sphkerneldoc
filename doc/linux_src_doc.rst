================================================================================
                             Source Code (aka autodoc)
================================================================================

Source code documentation from *autodoc*.

.. code-block:: bash

   ./scripts/autodoc.sh

The script ``autodoc.sh`` uses a modified version of ``kernel-doc`` to extract
the *doc-comments* in reStructuredText.
