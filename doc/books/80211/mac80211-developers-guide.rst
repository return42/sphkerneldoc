++++++++++++++++++++++
The mac80211 subsystem
++++++++++++++++++++++
    mac80211 is the Linux stack for 802.11 hardware that implements only
    partial functionality in hard- or firmware. This document defines
    the interface between mac80211 and low-level hardware drivers.

    If you're reading this document and not the header file itself, it
    will be incomplete because not all documentation has been converted
    yet.
