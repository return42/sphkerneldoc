
.. _API-input-grab-device:

=================
input_grab_device
=================

*man input_grab_device(9)*

*4.6.0-rc1*

grabs device for exclusive use


Synopsis
========

.. c:function:: int input_grab_device( struct input_handle * handle )

Arguments
=========

``handle``
    input handle that wants to own the device


Description
===========

When a device is grabbed by an input handle all events generated by the device are delivered only to this handle. Also events injected by other input handles are ignored while
device is grabbed.
