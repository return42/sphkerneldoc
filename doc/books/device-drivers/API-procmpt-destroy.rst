===============
procmpt_destroy
===============

*man procmpt_destroy(9)*

*4.6.0-rc1*

Tear down ``MPT_PROCFS_MPTBASEDIR`` entries.


Synopsis
========

.. c:function:: void procmpt_destroy( void )

Arguments
=========

``void``
    no arguments


Description
===========

Returns 0 for success, non-zero for failure.
