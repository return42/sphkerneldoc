===========
mptspi_init
===========

*man mptspi_init(9)*

*4.6.0-rc5*

Register MPT adapter(s) as SCSI host(s) with SCSI mid-layer.


Synopsis
========

.. c:function:: int mptspi_init( void )

Arguments
=========

``void``
    no arguments


Description
===========

Returns 0 for success, non-zero for failure.
