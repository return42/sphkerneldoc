eth_zero_addr
=============

*man eth_zero_addr(9)*

*4.6.0-rc1*

Assign zero address


Synopsis
========

.. c:function:: void eth_zero_addr( u8 * addr )

Arguments
=========

``addr``
    Pointer to a six-byte array containing the Ethernet address


Description
===========

Assign the zero address to the given address array.
