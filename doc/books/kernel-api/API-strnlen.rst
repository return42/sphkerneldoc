=======
strnlen
=======

*man strnlen(9)*

*4.6.0-rc5*

Find the length of a length-limited string


Synopsis
========

.. c:function:: size_t strnlen( const char * s, size_t count )

Arguments
=========

``s``
    The string to be sized

``count``
    The maximum number of bytes to search
