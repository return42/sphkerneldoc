======
memcmp
======

*man memcmp(9)*

*4.6.0-rc1*

Compare two areas of memory


Synopsis
========

.. c:function:: __visible int memcmp( const void * cs, const void * ct, size_t count )

Arguments
=========

``cs``
    One area of memory

``ct``
    Another area of memory

``count``
    The size of the area.
