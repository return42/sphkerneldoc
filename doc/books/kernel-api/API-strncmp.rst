strncmp
=======

*man strncmp(9)*

*4.6.0-rc5*

Compare two length-limited strings


Synopsis
========

.. c:function:: int strncmp( const char * cs, const char * ct, size_t count )

Arguments
=========

``cs``
    One string

``ct``
    Another string

``count``
    The maximum number of bytes to compare
