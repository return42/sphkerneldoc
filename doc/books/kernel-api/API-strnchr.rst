=======
strnchr
=======

*man strnchr(9)*

*4.6.0-rc5*

Find a character in a length limited string


Synopsis
========

.. c:function:: char * strnchr( const char * s, size_t count, int c )

Arguments
=========

``s``
    The string to be searched

``count``
    The number of characters to be searched

``c``
    The character to search for
