======
strcat
======

*man strcat(9)*

*4.6.0-rc5*

Append one ``NUL-terminated`` string to another


Synopsis
========

.. c:function:: char * strcat( char * dest, const char * src )

Arguments
=========

``dest``
    The string to be appended to

``src``
    The string to append to it
