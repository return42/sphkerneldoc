======
strcpy
======

*man strcpy(9)*

*4.6.0-rc5*

Copy a ``NUL`` terminated string


Synopsis
========

.. c:function:: char * strcpy( char * dest, const char * src )

Arguments
=========

``dest``
    Where to copy the string to

``src``
